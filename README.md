# aws-s3-backup
Example app to upload files in AWS S3

## Installation
````bash
npm install
````

## Configuration
create .env file with
````.env
BUCKET_NAME=  // AWS S3 bucket name
AWS_ACCESS_KEY= // AWS IAM user access_key
AWS_SECRET_ACCESS_KEY= // AWS IAM user secret_access_key
````
## Usage
````bash
node index.js
````
